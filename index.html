    Problem:

    Forms are password protected to ensure version control, but we wanted to be able to have the user add a page specifically for photos, without having to know the password to unlock the document.

    Solution:

    I was able to achieve this by saving a template of the photo page to each user's C: Drive.

    In the form, when the user clicked the command button, Word would:

    1. open the template document,
    2. copy its contents,
    3. unlock the form,
    4. paste the photo page to the location specified within the document,
    5. close the template, and
    6. re-lock the form with the password.
